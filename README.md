# NapCat Install

NapCat 安装脚本仓库。

脚本来源: https://nclatest.znin.net/NapNeko/NapCat-Installer/main/script/install.sh

## 使用方法

```bash
# 本地执行
bash install.sh

# 或在线执行
curl -fsSL https://nclatest.znin.net/NapNeko/NapCat-Installer/main/script/install.sh | bash
```

## 说明

本仓库存放 [NapCat-Installer](https://github.com/NapNeko/NapCat-Installer) 官方安装脚本，便于本地备份与部署使用。
